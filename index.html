Guía de Git con SourceForge

Aprende a utilizar Git con SourceForge paso a paso

Creación de Cuenta

  • Ingresar a SourceForge
  • Completar formulario de registro
  • Proporcionar nombre completo, usuario, clave, empresa, etc.

Creación de Proyecto

  • Asignar nombre al proyecto
  • Seleccionar herramientas a utilizar
  • Verificación con número de celular mediante PIN por SMS

Gestión de Miembros

  • Agregar colaboradores al proyecto
  • Asignar diferentes roles (administrador, desarrollador, etc.)
  • Control de acceso según roles asignados

Comandos Git para Subir Proyecto

1 Navegar al directorio del proyecto:

cd myproject

2 Inicializar repositorio Git:

git init

3 Agregar archivos al staging area:

git add .

4 Crear commit inicial:

git commit -a -m 'Initial commit'

5 Agregar repositorio remoto (usando SSH):

git remote add origin ssh://rutihsoc@git.code.sf.net/p/gitsource/code

6 Subir cambios al repositorio remoto:

git push -u origin master

Resolución de Problemas con SSH

En caso de tener problemas con SSH, se recomienda utilizar HTTPS:

git remote set-url origin https://git.code.sf.net/p/tu-proyecto/code

Luego volver a hacer push. Se abrirá una ventana emergente solicitando usuario y contraseña. Una vez insertados, el push se completará y los archivos estarán en el repositorio.
